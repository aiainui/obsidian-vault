# 英语发音笔记

> 来源：B站视频 - 专业美音纠音｜连读语调超细节分解

## 语调 Pattern

```
↗↘↗↘
升-降-升-降
```

这是英语口语中常见的**节奏模式**。

## 核心理解

| 符号 | 含义 | 表现 |
|------|------|------|
| ↗ (升调) | 轻声 → 重声 | 音调升高 + 音量增强 |
| ↘ (降调) | 重声 → 轻声 | 音调降低 + 音量减弱 |

## 适用场景

### 1. 列举
```
First, ↗ second, ↘ third, ↗ fourth, ↘
```

### 2. 对比
```
It's not good ↗, it's great ↘
```

### 3. 反问
```
You really think so ↗? ↘
```

### 4. 强调
```
That's the **most** ↗ **important** ↘ thing ↗
```

## 练习方法

1. **先找感觉**：↗↘↗↘ 反复读，形成肌肉记忆
2. **放在句子里**：别停单个词，放进完整句子练
3. **听 native speaker**：跟着视频里老师的语调模仿

## 总结

升调通常伴随音调升高+音量增强，降调则相反。反复练习形成语感。
