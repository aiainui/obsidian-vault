# 生成目录

扫描 vault 中的文件夹结构，生成两级目录（只显示到二级目录），并写入 README.md。

## 执行步骤

1. 扫描顶层的文件夹（一级目录）
2. 扫描每个一级目录下的子文件夹（二级目录）
3. 对于没有子文件夹的一级目录，列出该目录下的文件
4. 使用 Wiki-link 格式 `[[path]]`
5. 写入 README.md

## 输出格式

```markdown
# 我的 Obsidian 个人知识库

## 目录

### Agents
- [[Agents/xxx]] (二级目录)
- [[Agents/xxx]] (文件，如果没有子文件夹)

### Papers
- [[Papers]] (如果有子文件夹)
```

## 排序规则

- Agents 始终放在最前面
- 按字母顺序排列二级目录
