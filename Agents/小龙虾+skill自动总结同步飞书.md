## 文章概要

有没有人和作者一样，刷到公众号干货好文，第一反应就是转发给自己、赶紧收藏起来？结果存完就彻底忘在脑后，收藏夹里越堆越多，到最后全在里面默默吃灰。

作者用小龙虾（MCP）捣鼓出了一个超实用的小功能：
- 把文章链接丢进去
- 自动给文章做总结
- 自动打标签分类
- 直接把内容同步写到飞书文档里

---

## 实现步骤

### 1. 用 AskUserQuestion 采访需求
用 Claude Code 里的 AskUserQuestion 工具，让大模型化身采访者，帮你把没考虑到的细节、边界情况、异常处理全都补全。

### 2. 根据需求生成 SKILL.md
采访结束后，把 SPEC.md 丢给 Claude Code，让它生成 SKILL.md。

### 3. 获取飞书凭证
需要 4 个凭证：
- `FEISHU_APP_ID`
- `FEISHU_APP_SECRET`
- `FEISHU_SPREADSHEET_TOKEN`
- `FEISHU_SHEET_ID`

### 4. 添加到小龙虾使用
把凭证填到 SKILL.md 里，丢到 skills 目录，重启就能用了。

---

## SKILL 核心功能

1. **抓取链接页面内容**
2. **提取结构化信息**（摘要、关键词、分类）
3. **获取飞书 access_token**
4. **写入飞书电子表格**
5. **输出执行结果**

---

## 飞书表格列结构

| 列 | 字段 | 内容 |
|----|------|------|
| A | 标题 | 页面标题 |
| B | 链接 | 原始URL |
| C | 摘要 | 约100字中文摘要 |
| D | 关键词 | 3-5个关键词，逗号分隔 |
| E | 一级分类 | 技术/产品/设计/新闻/创业/工具/学习/其他 |
| F | 二级分类 | 细化分类标签 |
| G | 记录时间 | YYYY-MM-DD HH:MM:SS |
| H | 发布时间 | YYYY-MM-DD HH:MM:SS |
